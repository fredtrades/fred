# Dungeon Community Template – Design Brainstorm

## Three Stylistic Approaches

### Option A – Neon Dungeon
En mørk, gaming-inspireret æstetik med neon grønne accenter på sort baggrund. Inspireret af terminaler og hacker-kultur. Følelsesmæssigt: kraftfuld, eksklusiv, underground.
**Probability:** 0.07

### Option B – Tactical Dark
Militær/taktisk look med mørkegrå toner, skarp typografi og guld-accenter. Mere seriøs og professionel end gaming-vibes.
**Probability:** 0.02

### Option C – Cyber Vault
Futuristisk vault/bunker-tema med metallic overflader, glitch-effekter og cyan-accenter. Sci-fi inspireret.
**Probability:** 0.01

---

## Chosen Approach: **Neon Dungeon** (Option A)

### Design Movement
Dark Web / Terminal Aesthetic – inspireret af hacker-kultur, gaming UI og underground communities.

### Core Principles
1. Maksimal kontrast: rent sort baggrund mod neon grøn
2. Uppercase typografi med stor skriftstørrelse for dominans
3. Dashed/dotted borders som reference til terminal/Discord UI
4. Minimal men præcis brug af farve – grøn er hellig

### Color Philosophy
- Background: `#080808` – næsten rent sort, dybt og immersivt
- Neon Green: `#39ff14` – elektrisk, levende, uundgåelig
- Gold/Olive: `#c8a84b` – sekundær accent til headings og badges
- White: `#f0f0f0` – primær tekstfarve
- Dark Card: `#111111` / `#1a1a1a` – kortbaggrunde
- Grøn glow: `box-shadow: 0 0 20px rgba(57, 255, 20, 0.3)`

### Layout Paradigm
Asymmetrisk full-width sektioner med vekslende layouts. Hero er centreret med massiv typografi. Cards-sektioner bruger grid. Giveaway-sektion er split 60/40. Footer er 3-kolonne.

### Signature Elements
1. Dashed neon grøn border på CTA-knapper (som Discord-UI)
2. Glowing neon grøn tekst på primære headings
3. Uppercase condensed display font til alle sektionsoverskrifter

### Interaction Philosophy
Hover-effekter med neon glow. Knapper skalerer subtilt. Smooth scroll. Cards løfter sig med grøn kant ved hover.

### Animation
- Fade-in på sektioner ved scroll
- Knapper: scale(0.97) on active, 150ms ease-out
- Cards: translateY(-4px) + green border glow on hover, 200ms
- Hero titel: staggered entrance animation

### Typography System
- Display: `Bebas Neue` (Google Fonts) – ultra-bold, condensed, uppercase
- Body: `DM Sans` – clean, moderne, læsbar
- Nav: `DM Sans` medium, uppercase, letter-spacing

### Brand Essence
Den ultimative community-hub for seriøse tradere – eksklusiv, troværdig, og altid i bevægelse.
Personality: **Dominerende · Autentisk · Underground**

### Brand Voice
Headlines: direkte, kraftfulde, ingen fluff. "WIN PROP FIRM ACCOUNTS" ikke "Discover opportunities".
CTAs: action-orienterede. "Enter the Dungeon" ikke "Get started".
Microcopy: præcis og ærlig.

### Wordmark & Logo
Et stiliseret 'D' formet som et dungeon-skjold eller portal, i neon grøn på sort baggrund.

### Signature Brand Color
Neon Electric Green `#39ff14`
